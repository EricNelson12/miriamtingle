# Hello!

These are the high level steps to update this simple HTML and CSS website.

## What you’ll need
- A free GitHub account
- [Visual Studio Code](https://code.visualstudio.com/) (for editing)
- Git or GitHub Desktop (for uploading)

## Steps
1. **Clone the repo** – Use Git to make a copy of the repository on your computer.  
2. **Open and edit** – Use Visual Studio Code to open the folder and make your changes.  
3. **Upload to GitHub** – Use Git or GitHub Desktop to publish your edits.
